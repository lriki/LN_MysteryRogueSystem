実装設計-壺
==========

- ItemBehavior に加え、PotBehavior を持つ。
- PotBehavior はアイテムの配列を管理するだけで、出し入れといった Command に応答する機能は無い。
  というのも、保存の壺以外はアイテムを取り出すことはできないし、手動で入れることすらできる壺は限られている。
- 壺の効果は Ability にしない。
